确定有限自动机（DFA，deterministic finite automaton）

```
M=(S,∑, f, S0, F)
解释：S有穷状态集，∑输入字母表（有穷），f状态转换函数（单值部分映射）， S0唯一初态，F终态集（可空）
什么叫a为DFAM所识别(接收)
```

非确定有限自动机(NFA，)

```
M=(S,∑, f, S0, F)
解释：S有穷状态集，∑输入字母表（有穷），f状态转换函数（）， S0非空初态，F终态集（可空）
```

NFA相对于DFA的区别

```
1.可以有多个初态；
2.弧上的标记可以是S*中的一个字(甚至可以是一个正规式)，而不一定是单个字符；
3.同一个字可能出现在同状态射出的多条弧上。
DFA是NFA的特例。
```
